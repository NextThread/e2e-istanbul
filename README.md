1. run `npm install -g grunt-cli`
1. run `npm install -g jasmine-node`
1. Download chrome webdriver: http://chromedriver.storage.googleapis.com/index.html
    - put in in your PATH
1. Clone this repo
1. cd into this directory
1. run `npm install`
1. `grunt dev`
1. `jasmine-node specs/test-spec.js`
1. `grunt report`
