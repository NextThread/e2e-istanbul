1. run `npm install -g grunt-cli`
1. run `npm install -g jasmine-node`
1. Download firefox webdriver: https://github.com/mozilla/geckodriver/releases/
    - put in in your PATH
1. Clone this repo
1. cd into local repo directory
1. run `npm install`
1. `grunt dev`
1. `jasmine-node specs/test-spec.js`
1. `grunt report`
